This project uses the Queens Sales data from the link (http://www1.nyc.gov/home/search/index.page?search-terms=Rolling%20sales%20update&start-number=10)

Purpose
- The purpose of this study is to use Sales Data from Queens to see how Square Foot and Price are related in this neighborhood. The data is raw and needs to be cleaned. Then it can be analyzed. 

The Data File
- Removes Outliers
- Removes Missing Values
- Change variable names
- Take log of large and skewed data

The Analysis File
- A plot of price by the square foot

The Paper File
- Explanation of the analysis from the plot
- What we can and cannot conclude from this analysis
